# 概要 
数据集包含 1,157名棒球手的数据，包括他们的用手习惯（左手还是右手）、身高（英寸）、体重（磅）、击球率和全垒得分。目标是创建一个可视化图表，展示这些球手的表现差异。

# 设计 
* 图表类型：气泡图
* 维度：击球率或全垒
* 互动：点击图例可以查看不同利手成绩
* 观点：左利手和右利手的成绩没有明显差异，全垒得分较高（>350）的人没有双利手的
# 反馈 
* 反馈一：布局太靠左
改进：整体右移
* 反馈二：缺少标题
改进：添加标题和标题样式
* 反馈三：点击后反应太慢
改进：减少duration时间
* 反馈四：第二次点击不会重新出现
改进：增加重画数据范围
